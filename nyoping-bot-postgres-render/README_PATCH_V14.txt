뇨핑봇 패치 V14

수정 내용
- 반응 차단 리스너 구조를 다시 잡아 실제 반응 차단/보정이 동작하도록 수정
- 반응 역할 규칙 매칭 fallback 강화(특히 반응 취소 시 no rule matched 완화)
- 출석 명령어가 내부 예외로 전체 실패하지 않도록 보강
- 환영 이미지 미리보기: 업로드한 배경 이미지 원본 크기 유지
- 환영 이미지 렌더링: 한글 폰트 우선 사용
- 환영 이미지: 텍스트 2, 텍스트 3 추가 지원
- 채널 제어: 초대 링크 금지 / 봇 전용 채널을 리스트 선택형으로 변경
- 왼쪽 메뉴 재구성: 레벨 설정 / 자동 인사 / 채널 제어 / 반응 / 서버 랭킹

적용
- Render: nyoping-bot-postgres-render 전체 덮어쓰기 후 재배포
- weirdhost: nyopingbot 폴더 전체 + requirements.txt 반영 후 재시작
- 반응 역할 / 반응 차단 규칙은 한 번 다시 저장 권장
