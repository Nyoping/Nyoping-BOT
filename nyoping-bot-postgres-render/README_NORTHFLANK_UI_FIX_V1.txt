Northflank UI 복구 핫픽스 v1

이번 패치
- /admin 서버 선택을 ID 직접 입력 대신 서버 목록 드롭다운으로 변경
- 왼쪽 메뉴 클릭 시 실제로 보이는 카드 묶음을 정리
- 레벨 설정 / 자동 인사 / 채널 제어 / 반응 / 서버 랭킹 메뉴 동작 정리
- 빠른 작업/레벨 역할 규칙 카드는 기본 상태에서 중복 노출되지 않도록 숨김 처리 후 메뉴 클릭 때만 표시

적용 방법
- Northflank 대시보드 서비스 기준으로 nyoping-bot-postgres-render 전체를 GitHub에 덮어쓰기
- 커밋/푸시 후 Northflank 재배포
- 로그인이 되면 /admin에서 서버 목록 드롭다운이 보이는지 확인

주의
- 이 패치는 UI 복구용이라 봇/DB 로직은 건드리지 않음
