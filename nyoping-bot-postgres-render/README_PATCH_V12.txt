뇨핑봇 패치 V12 - 환영 이미지 업로드/미리보기 편집기

이번 패치 내용
1. 대시보드에서 배경 이미지를 파일 첨부로 업로드 가능
2. 업로드한 이미지는 Postgres에 저장되고, 내부 URL(/media/...)이 자동 생성됨
3. 환영 이미지 미리보기 편집기 추가
   - 선택한 유저의 닉네임/아바타로 미리보기
   - 아바타 위치 드래그 가능
   - 아바타 크기(우하단 핸들) 조절 가능
   - 텍스트 위치 드래그 가능
   - 텍스트 박스 너비(우하단 핸들) 조절 가능
4. 유저 캐시에 avatar_url 저장해서 미리보기에서 디스코드 프로필 사진 사용

적용 방법
- Render: nyoping-bot-postgres-render 전체 교체 후 재배포
- weirdhost: nyopingbot/ 전체 + requirements.txt 교체 후 재시작

주의
- 미리보기는 대시보드에서 선택한 샘플 유저 기준입니다.
- 실제 환영 메시지는 입장한 멤버의 닉네임/아바타로 적용됩니다.
- 이미지 업로드는 8MB 이하 권장
