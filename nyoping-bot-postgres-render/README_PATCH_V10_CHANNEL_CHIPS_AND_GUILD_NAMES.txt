v10 수정 내용

1) 채널 제어 > 초대 링크 금지 채널 칩 복구
- 서버 렌더링으로 현재 설정된 채널 칩을 먼저 보여주도록 수정
- JS가 채널 ID 목록을 더 안정적으로 파싱하도록 보강
- 채널 picker item에 data-channel-id를 넣어서 선택 칩 매칭 복구

2) 로그인 후 서버 선택 목록 개선
- 서버 이름이 비어 있거나 숫자(ID)처럼 보이면 Discord API / 캐시 기준으로 실제 서버 이름 재조회
- 목록 표시를 항상 '서버이름 · 서버ID' 형태로 통일

적용 후 기대 결과
- 채널 제어 메뉴에서 기존에 선택했던 채널 칩이 다시 보임
- 서버 선택 드롭다운에서 서버 이름과 서버 ID가 같이 표시됨
