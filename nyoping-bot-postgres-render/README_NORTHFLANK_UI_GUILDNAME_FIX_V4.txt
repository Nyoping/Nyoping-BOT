뇨핑봇 Northflank 패치 V4

수정 내용
- 서버 선택 드롭다운에 guild 이름이 표시되도록 guilds_cache 테이블 추가
- 봇이 guild 이름을 주기적으로 DB에 동기화하도록 수정
- 대시보드가 guilds_cache를 우선 읽도록 수정
- '빠른 작업' 제목을 '레벨 관리 작업'으로 변경해 레벨 설정 그룹에 통합
- 레벨 설정 메뉴에서 설정/관리 작업/레벨 역할 규칙이 같이 보이도록 유지

적용
- Northflank 봇 서비스와 대시보드 서비스 둘 다 같은 저장소 최신 커밋으로 재배포
- guild 이름은 봇이 한 번 캐시 동기화를 돌고 나면 표시됨 (보통 로그인 직후 ~ 10분 이내, 또는 재시작 직후 1회)
