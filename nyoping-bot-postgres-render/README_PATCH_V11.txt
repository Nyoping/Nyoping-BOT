뇨핑봇 패치 V11

이번 패치 주요 내용
1) /프로필 오류 수정
- InteractionResponded 오류 수정
- /프로필로 다른 유저도 조회 가능 (옵션 user)

2) 반응 역할
- 반응 취소 시 역할 역적용
  * add 규칙으로 받은 역할은 취소 시 제거
  * remove 규칙으로 뺀 역할은 취소 시 다시 부여

3) 반응 차단
- 반응 제거 자체를 같은 사용자로 강제 복구하는 것은 Discord API 구조상 불가
- 대신 마지막 이모지가 사라지는 경우를 대비해 봇 반응으로 다시 남겨두는 best-effort 처리 추가

4) 자동 환영/퇴장
- 채널 지정
- 메시지 템플릿 변수: [user] [server] [inviter] [discord] [reason]
- 환영 이미지: 배경 URL, 아바타 위치/크기/모양, 텍스트 위치/크기/색/정렬/폰트

5) 채널 제어
- 특정 채널 초대 링크 금지
- 특정 채널 봇/명령어 전용

6) 음성/랭킹
- AFK 채널 자동 퇴장
- 통화 종료 DM XP 요약
- 특정 채널 고정 랭킹 갱신(5분 간격)

7) 공개 알림
- notify_channel_id 설정 시 레벨업/레벨 역할 변경 공개 알림

주의
- 초대 링크 금지, 환영/퇴장, 초대자 감지, 강퇴 판정은 봇 권한 상태에 따라 일부 제한될 수 있음.
- 환영 이미지는 URL 기반 배경 이미지를 사용합니다.
