뇨핑봇 v1.6 패치 내용

- 일반 사용자도 비밀번호 없이 접속 가능한 사용자 페이지 추가
- 경로: /user
- 사용자 페이지에서는 서버 선택 후 유저 프로필, 현재 랭킹, 출석 횟수, 연속 출석, 오늘 통화 XP, 서버 랭킹 조회 가능
- 관리자 비밀번호는 그대로 유지
- 관리자 설정 페이지(/admin)는 여전히 관리자 로그인 필요
- 메인 화면(/)에 '일반 사용자 입장' 버튼 추가

적용 후 확인:
1. / 에서 일반 사용자 입장 버튼이 보이는지
2. /user 에서 서버 선택이 되는지
3. 유저 선택 후 프로필/랭킹이 보이는지
4. /admin 은 여전히 관리자 로그인 없이는 못 들어가는지
