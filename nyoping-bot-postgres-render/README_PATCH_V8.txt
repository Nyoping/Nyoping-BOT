이번 수정:
1) 반응 역할 추가/제거
- 현재 규칙에서 이모지 표시 수정
- 삭제 버튼 422 오류 수정
- 저장 직후에도 규칙이 바로 적용되도록 DB를 즉시 조회하게 수정

  봇이 관리 못 하는 다른 역할 때문에 전체 실패하는 경우를 줄임

2) 연속 출석 보너스 테스트 모드
- 출석 제한 OFF 상태에서도 같은 날 여러 번 출석하면 streak가 1씩 올라감
- 따라서 연속 출석 보너스도 같은 날 반복 테스트 가능
