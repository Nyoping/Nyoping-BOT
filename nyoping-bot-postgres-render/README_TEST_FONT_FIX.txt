테스트용 환영 메시지 폰트 수정

원인:
- 대시보드 테스트 이미지 생성 함수에서 한글이 포함되면 사용자가 고른 폰트를 무시하고
  강제로 "korean" 폰트 그룹으로 덮어쓰고 있었습니다.

수정:
- 사용자가 고른 font_name을 우선 사용하도록 변경
- _safe_font()의 fallback 체인만 이용해 한글 글리프가 없을 때 대체 폰트를 찾도록 정리

적용:
- nyoping-dashboard만 재배포하면 테스트용 메시지 이미지 폰트가 바뀝니다.
- 실제 입장 환영 이미지는 이미 선택한 폰트를 사용하고 있어서 봇 재배포는 필수 아님
