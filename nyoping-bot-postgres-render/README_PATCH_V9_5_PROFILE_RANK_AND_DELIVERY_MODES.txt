V9.5 수정 사항

1) /프로필 명령어
- 현재 랭킹 등수 표시 추가
- 형식: 현재 랭킹: N위 / 총 M명

2) 안내 방식 설정 추가
대시보드 > 레벨 설정 > 안내 방식

추가된 설정:
- /출석 결과: 본인만 보기 / 공개 알림 채널 / DM
- /프로필 결과: 본인만 보기 / 공개 알림 채널 / DM
- /랭킹 결과: 본인만 보기 / 공개 알림 채널 / DM
- 레벨업 자동 알림: 공개 알림 채널 / DM / 끄기
- 통화 XP 자동 알림: DM / 공개 알림 채널 / 끄기

3) 공개 알림 채널
- 기존 notify_channel_id(공개 알림 채널)를 공용으로 사용
- 공개 선택 시 이 채널로 보냄

4) 참고
- 자동 알림(레벨업, 통화 XP)은 Discord 구조상 본인만 보기(ephemeral)가 불가능
- 그래서 자동 알림은 DM / 공개 / 끄기 방식으로 구현됨

적용 방법
1. GitHub 저장소를 이 전체 프로젝트 ZIP 기준으로 덮어쓰기
2. Northflank에서 nyoping-dashboard 재배포
3. Northflank에서 nyoping-bot 재배포
4. 대시보드에서 해당 서버 설정 저장 1회
5. /프로필, /랭킹, /출석, 레벨업, 통화 XP 알림 테스트
